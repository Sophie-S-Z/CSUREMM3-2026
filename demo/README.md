# Demo (optional)

This folder is reserved for an optional toy/synthetic-data demonstration —
something that illustrates the *shape* of the modeling approach (e.g., on a
small synthetic network) without using real study data, tuned parameters, or
the full private pipeline.

Nothing has been added yet. Only populate this folder with code that is safe
to make fully public: no real data, no calibrated parameters from the actual
study, no internal utilities shared with the private working repo.
